Geo Chart Demo

Customer Demographics - Location

Viz Type:Map with Overlays
Data:Dummy Data
Show Me:Where are my customers?
Description:An interactive and obvious way to display the geographic dispersion of customers. The size of the circle represents the number of customers in a zip code and the location of the circle represents the location of the customer. The map is scrollable and zoomable just like a user would expect from a Google map.
Visualization Justification:A map is the most obvious way to display geographic data. There is no learning curve to viewing a map, everyone already knows how to read one, yet it still provides rich, contextual information at a glance.

Regional Performance

Viz Type:GeoChart with Markers
Data:Dummy Data
Show Me:How performance differs by region
Description:A simple and clean view of regional data, a GeoChart allows for quick comparisons of pre-defined regions. The size and color change as values increase. In this case, bigger and more blue means higher sales. Hover over the markers to see the region name and value.
Visualization Justification:Simpler to read than a choropleth map or a heat map, a GeoChart uses leverages the user's existing knowledge of areas to overlay information. While not particularly information rich, a GeoChart can be useful for exploratory analysis, exposing relationships that would otherwise be missed in a tabular display.
